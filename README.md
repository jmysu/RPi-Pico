### Interesting projects based on RPi-Pico<br>
 
My UNO bread-board w/ RPi-Pico & Arduino-Nano!
With some features:
 - Neopixel RGB LEDs.
 - Shared I2C, SPI, UART, A/D & SD card.
 - 5V/3V3 selectable.
 - SPI LCD modulable compatible w/ touch.
 - Arduino-Uno/Mega compatible size/Holes.
